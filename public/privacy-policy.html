Privacy Policy

Overview

This Privacy Policy describes how the Skyhawk Squadron Automation Script application handles data.

Data Collection

This application stores limited authentication credentials provided by authorized users in order to access Civil Air Patrol administrative systems.

How Data Is Used

  • Authenticate authorized users
  • Retrieve squadron administrative data
  • Display cadet achievement tracking information

Data Sharing

This application does not sell, share, or distribute user data to third parties.

Security

Credentials are stored securely using platform-provided storage mechanisms and are only used for authorized API access.

Contact

For privacy concerns, contact the application administrator for Skyhawk Squadron.
